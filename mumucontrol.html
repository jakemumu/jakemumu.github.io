About this project

Mumu control is a custom midi controller built using an arduino
The controller was preseted at the 2015 Digital Arts expo as a interactive audio visual controller. The controller was mapped to control an ableton session as well as a particle system, as a user played with the controller they could hear changes in the sound with visuals synced to the current paramters and buttons pressed.
