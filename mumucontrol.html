About this project

A custom midi controller built with arduino.

This project served as an exploration into the world of electrical engineering, physical computing, and rapid prototyping.

Custom circuitry was developed in order to create an interface which interpreted data from a gyroscope, accelerometer, and an ultrasonic sensor, as well as potentiometers, push buttons, and switches. The data from these components was sent to a computer via serial, and interpreted by a chuck script which converted the messages to midi. A custom case was fabricated from laser cut acrylic.

Presented at the 2015 Digital Arts expo as a interactive installation, the controller, mapped to both audio and visual, invited the user to play with sound and space.
